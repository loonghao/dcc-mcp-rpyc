Installation
============

Stable release
-------------

To install DCC-MCP-IPC, run this command in your terminal:

.. code-block:: console

    $ pip install dcc-mcp-ipc

Or with Poetry:

.. code-block:: console

    $ poetry add dcc-mcp-ipc

This is the preferred method to install DCC-MCP-IPC, as it will always install the most recent stable release.

From sources
-----------

The sources for DCC-MCP-IPC can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/loonghao/dcc-mcp-ipc

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/loonghao/dcc-mcp-ipc/tarball/main

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ poetry install


.. _Github repo: https://github.com/loonghao/dcc-mcp-ipc
.. _tarball: https://github.com/loonghao/dcc-mcp-ipc/tarball/main
